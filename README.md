# Smart Timetable Generator
An automated scheduling platform built on `bolt.new` that simplifies institutional planning. 

This application eliminates scheduling conflicts by automatically mapping: 
* teachers
* classrooms
* classes
  
into an optimised weekly timetable.🚀


## Features
* **Teacher Management:** Input faculty details, subjects, and availability.
* **Classroom Allocation:** Manage physical rooms, capacities, and resources.
* **Class & Section Creation:** Set up student groups and required weekly subject hours.
* **Smart Automation:** Generate conflict-free schedules instantly with a single click.
* **Interactive Grid:** View and export timetables easily with colour-grading.
